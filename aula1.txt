# Python:
Referência: https://www.python.org/
Python é uma linguagem de programação que te permite integrar sistemas de uma forma mais eficiente. Será utilizado para aprender e desenvolver, desde lógica até aplicações mais avançadas.

# Git:
Referência: https://git-scm.com/
Git é uma plataforma open-source, grátis, que te permite controlar sistemas de forma escalável com velocidade e eficiência. Será utilizado para o versionamento e atualização de nossas aplicações.

# VSCODE:
Referência: https://code.visualstudio.com/
VSCODE é uma interface de desenvolvimento integrada desenvolvida pela Microsoft que te permite criar e editar código de forma gratuíta. É um dos IDEs mais usados no mundo, por ser interativo, simples e por possuir suporte a várias linguages. Será utilizado para desenvolver nossas aplicações.

# Sqlite browser:
Referência: https://sqlitebrowser.org/
Sqlite é uma plataforma que te permite projetar, criar e gerenciar bancos de dados. Será utlizado para gerenciar os bancos de dados de nossas aplicações.

# Navegadores (Chrome):
Referência: https://www.google.com/intl/pt-BR/chrome/
Chrome é um navegador web gratuíto que será utilizado para testar as aplicações feitas durante as aulas. Será utlizado para interagir e testar nossas aplicações.
